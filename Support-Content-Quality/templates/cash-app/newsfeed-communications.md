*Source: https://docs.google.com/document/d/14sR8c04NRTkHLKb-C_Qgs0FhdshG6I7zo8L8-B4lHQA/edit*

## Overview

This is a template for CCO Comms sent via the [CCO Newsfeed](https://go/cconews). Copy this document to draft individual comms.

🚨 **Please do not copy this to create a new template for your team.** 🚨 This is a living document that changes every now and then. Instead, link to this document within your own team resources, if needed.

## Define Audience

ℹ️ While the template includes some tips below, please see [Target Audience Guidelines](https://docs.google.com/document/d/12EQSqQLIVMiCBN1Pi-bG1LN_tZnW05Kjtyh0n1OE0_U/edit?usp=sharing) for the full set of instructions.

| Field | Template |
|-------|----------|
| **Worker classification** | ℹ️ This defines whether or not the comm will be visible to Partner workers within the other groups you select later.<br><br>**All:** Partner and Internal<br>**Full-Time:** Internal only<br>❌ Contingent (excludes BPO): Don't use<br>❌ BPO: Don't use |
| **Exempt/Non-exempt targeting** | **Both**<br>❌ Exempt: Don't use<br>❌ Non-Exempt: Don't use |
| **Location** | Don't use |
| **Job Families** | Don't use |
| **Organization (team code)** | ℹ️ Team Codes are a common way to target communications to specific groups within Cash App. Depending on the communication, you may use zero, one, or several Team Codes.<br><br>Just need to send the comm to All CCO? See [these instructions](https://docs.google.com/document/d/12EQSqQLIVMiCBN1Pi-bG1LN_tZnW05Kjtyh0n1OE0_U/edit?tab=t.ymmnv29wum#bookmark=kix.r5e6wqr7ibpt).<br><br>**Primary Team Code(s):**<br><br>**[Secondary Team Code(s)](https://docs.google.com/document/d/12EQSqQLIVMiCBN1Pi-bG1LN_tZnW05Kjtyh0n1OE0_U/edit?tab=t.ymmnv29wum#bookmark=kix.4k2ao7983pgv):**<br><br>Secondary Team Codes should be used any time you're not using the [All CCO](https://docs.google.com/document/d/12EQSqQLIVMiCBN1Pi-bG1LN_tZnW05Kjtyh0n1OE0_U/edit?tab=t.ymmnv29wum#bookmark=kix.r5e6wqr7ibpt) instructions. |
| **Manager/HRBP targeting** | ℹ️ Use this field if your comm is intended to only reach managers/leads.<br><br>**All Employees** (Will appear when field is blank - use this most of the time)<br>**Managers of Full Time Employees**<br>❌ Managers of Contingent Workers: Don't use<br>❌ HRBPs: Don't use |
| **Manager roll-up** | ℹ️ Use this field if you need to filter by a specific manager and their reporting line. This will only include the selected manager(s) + everyone under them.<br><br>**Blank**<br>**Use:** |

## Who Else Should Have Access to View This Article?

ℹ️ For visibility into CCO updates, it's critical to expand viewing to all of Block for every communication. Without this step, only those in the target audience settings selected above can open links to the post.

⚠️ Skip this step for sensitive/confidential updates.
