## 🔄 Template Maintenance

### When to Update Templates
- Product features or flows change
- Style guidelines are updated
- User feedback indicates confusion
- New tools or systems are introduced

### How to Propose Changes
1. Test proposed changes with real content
2. Ensure alignment with [style guidelines](../style-guides/)
3. Document the rationale for changes
4. Update related examples and documentation
